# Reasoning Memory

Critical-thinking memory for "should we do this" signals.

Capture high-level judgement history:
- challenged decisions
- discarded alternatives
- invalidated assumptions
- risk signals that actually materialized

## Entry Template

```markdown
## [YYYY-MM-DD] [decision-title]
- Initial recommendation: [what was proposed first]
- Main counter-argument: [strongest dissent]
- Alternatives explored: [A, B, C]
- Final choice: [selected option]
- Risk accepted: [explicit trade-off]
- What proved wrong/right later: [runtime feedback]
- Rule update for future decisions: [new threshold or trigger]
- Linked artifacts: [spec/ticket/adr/incidents]
```
