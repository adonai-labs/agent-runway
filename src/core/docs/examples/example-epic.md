# Example Epic (Multi-tenant Isolation)

## Objective
Deliver enforceable tenant isolation for scoped resources.

## Business Outcome
- Reduce cross-tenant exposure risk.
- Increase enterprise compliance readiness.

## Included Work
1. API tenant context + fail-closed checks
2. Application-level tenant propagation
3. Repository tenant filtering contracts
4. Integration tests + denied-access audit logs

## Exit Criteria
- Cross-tenant access blocked across covered entities.
- Integration tests pass for allow/deny scenarios.
